# Telgrambot
Simple Telegram Bot for Network Automation
I just built a Telegram bot that lets me run Cisco router commands from my phone — here's exactly how I did it.In this article, I'm Just trying to explain a very basic way on how you can use python and Telegram to create a chat bot. This particular process requires manual intervention and it doesn't use any LLM's or AI models

Cisco IOL routers running in EVE-NG (R1 and R2 with OSPF)
Python + Paramiko for SSH automation
python-telegram-bot to expose commands as chat messages
A single bot.py file — no frameworks, no complexity
